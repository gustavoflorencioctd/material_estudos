DDL (Data Definition Language) - São comandos que servem para a definição extruturação do banco de dados.

	- CREATE DATABASE <name>: Cria um banco de dados;	
	- CREATE TABLE <name>: Cria uma tabela;

DML (Data Manipulation Language) - São comandos que servem para a manipução dos dados nas tabelas.

	- INSERT INTO <name table> VALUES: Inseri valores nas linhas das tabelas;
